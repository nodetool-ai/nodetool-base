# nodetool.nodes.lib.yt_dlp

## VideoDownload

Download a video from a URL using [yt-dlp](https://github.com/yt-dlp/yt-dlp).

Use cases:
- Save videos for offline processing
- Fetch clips for video editing workflows
- Archive online content
